# 3分钟快速自由上网

1. [下载ClashVergeRev（推荐）](https://www.clashverge.dev/install.html)
    复制页面内的安装包下载地址，如果加载不出到[GithubRelease](https://github.com/clash-verge-rev/clash-verge-rev/releases/)复制【下载】中x64的地址

2. 任选其一，粘贴刚才复制的链接，加速下载安装包
    [加速工具1](https://ghp.ci/) [加速工具2](https://gh-proxy.com/) [加速工具3](https://www.7ed.net/gitmirror/hub.html)

3. 安装并运行

4. 在下面的免费分享网站中任选一个，找到一个订阅地址并复制

| 网站                                 | 地址                                                         |
| ------------------------------------ | ------------------------------------------------------------ |
| 液泡节点（暂不可用）                 | ~~https://magic.vacuole.top/fc65e21e-7dc0-4aa1-a453-bf10a41b808f~~ |
| V2RayShare                           | https://v2rayshare.com/                                      |
| 长风（需要向下滚动，不要复制base64） | https://www.cfmem.com/search/label/free                      |

5. 在ClashVerge中使用链接导入配置
6. 在`设置`->`系统设置`中打开`服务模式`和`系统代理`
7. 回到`代理`选项卡中测度并选择节点

> **用完及时关掉**
